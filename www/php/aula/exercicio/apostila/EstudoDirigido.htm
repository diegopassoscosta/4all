Etapa 0 – Banco de dados

 

1) Crie um banco de dado com nome Exer_sys01.

2) Execute o script de criação das tabelas ¨cidade¨ e ¨estado¨. (Exer_sys01.sql)

3) Crie uma pasta chama ¨libary¨.

4) Crie um arquivo php para realizar conexão com o base de dados →  Exer_sys01. Salve o arquivo com nome conexao.php.

 

 

Etapa 1 – Módulo [Estado]

 

5) Crie um arquivo php para realizar listagem de todos os dados da tabela ¨estado¨ salve-o com o nome estado_lista.php. Não esqueça de realizar o require_once(importação) do arquivo de conexão com o banco de dados.

 

 

Obs.: Na listagem não esqueça de realizar a coluna <td>Opção</td> e criar a opção de deleçao enviando um link para estado_deletar.php

 

6) Edite o estado_lista.php. Adicione um formulário de inserção para adicionar novos estados.

 

Obs.: não esqueça de adicionar o action apontando para o arquivo estado_inserir.php.

7) Crie um arquivo php para realizar inserção de dados na tabela ¨estado¨, salve-o com o nome estado_inserir.php. Não esqueça de realizar o require_once(importação) do arquivo de conexão com o banco de dados.

Obs.: para debugar/testar a query que será executado no banco de dados realize o comando exit() como mostra o trecho comentado.

 

8) Crie um arquivo php para realizar deleçao de dados na tabela ¨estado¨, salve-o com o nome estado_deletar.php. Não esqueça de realizar o require_once(importação) do arquivo de conexão com o banco de dados.

Obs.: para debugar/testar a query que será executado no banco de dados realize o comando exit() como mostra o trecho comentado.

 

Etapa 2 – Módulo [Cidade]
